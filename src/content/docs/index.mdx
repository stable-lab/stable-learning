## Speculative Decoding

How modern LLM serving generates 3–6× faster without changing a single output token — from the lossless acceptance rule to EAGLE, DFlash, and DSpark.

**Prerequisites**: transformer basics (attention, KV cache, logits). Pairs well with the RL track's LLM context but does not require it.

<LinkCard title="1. Fundamentals" description="Watch a draft model race the target model live, prove the two-line rule that makes it lossless, and price the speedup." href="specdec/01-fundamentals/" />
<LinkCard title="2. The EAGLE Line" description="Draft from the target's own features, grow confidence-priced draft trees, and the training-time test that unlocked data scaling." href="specdec/02-eagle/" />
<LinkCard title="3. Parallel Drafting" description="DFlash drafts whole blocks with a diffusion model; DSpark meters verification per request under real serving load." href="specdec/03-parallel-drafting/" />
